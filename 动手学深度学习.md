# 动手学深度学习

## 安装

```powershell
#安装MiniConda 并创建环境
conda create --name tensorflow python=3.9 -y
conda activate tensorflow
#安装CUDA cuDNN
#安装 tensorflow
pip install tensorflow -i https://pypi.tuna.tsinghua.edu.cn/simple/
pip install tensorflow-probability -i https://pypi.tuna.tsinghua.edu.cn/simple/
#安装d2l包
pip install d2l -i https://pypi.tuna.tsinghua.edu.cn/simple/
#cd到对应目录 启动 jupyter notebook
jupyter notebook
```
